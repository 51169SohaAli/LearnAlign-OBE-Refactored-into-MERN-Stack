# 📌 OBE Management System

## 📖 Overview
A web-based Outcome-Based Education (OBE) Management System designed to streamline academic processes such as course management, CLO mapping, assessments, and student performance tracking.

The project was initially built using HTML, CSS, JavaScript, PHP, and MySQL, and is currently being converted into a MERN stack (MongoDB, Express, React, Node.js) for improved scalability and modern architecture.

---

## 🚀 Features

### 🧑‍🎓 Authentication System
- User login system for secure access

### 📊 OBE Core Interface
- Course and assessment management
- CLO mapping structure
- Dashboard for system overview

### 🛠 In Progress (MERN Migration)
- Backend API development (Node + Express)
- React-based frontend upgrade
- MongoDB integration
